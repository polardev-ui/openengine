# Beta - Enhanced Realistic Voice

Enhanced version using Edge TTS with Aria Neural voice for Advanced Neural-quality speech.

## Files

- `voice_assistant.py` - Main beta assistant
- `realistic_voice.py` - Enhanced Edge TTS engine with Aria + styles
- `human_voice.py` - Human personality engine

## Run

```bash
cd beta
python3.11 voice_assistant.py
```

## Features

- **Aria Neural voice** - Most advanced conversational voice
- **6 speaking styles** - chat, cheerful, excited, friendly, empathetic, customerservice
- **Enhanced SSML** - Natural prosody and expressiveness
- **Sesame AI quality** - Natural, human-like conversation

## Compare with Production

Production uses AvaNeural (good), Beta uses AriaNeur (excellent with styles).
